# Code Review Rules

## JavaScript/React
- Usar componentes funcionales de React
- Usar const/let, nunca var
- PreferirArrow functions para callbacks
- No usar any types en TypeScript

## Estructura
- Usar extensiones .jsx para componentes React
- Mantener importaciones organizadas
- No hardcodear valores de configuración

## Buenos hábitos
- Usar async/await en lugar de then/catch
- Manejar errores apropiadamente
- Usar nombres descriptivos para variables
